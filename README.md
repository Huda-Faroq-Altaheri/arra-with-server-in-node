 ## Installation

To deploy this project run

```bash
  npm install 
```

## *CRUD* operations in node

 Api  `getAllUser` for get all users.

 Api `addUser` for add new user.

 Api `update` for update user by id.

 Api `delete` for delete user by id.

 Api `getUserById` for get user by id.

 Api `searchByKey` for search by name.
 
 Api `getAllUserReveres` for get all users *reversed*.
